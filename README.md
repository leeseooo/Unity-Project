■게임 소개
눈송은 방에서 일어나 지하철을 타고 대학가를 지나 학교에 가야한다... 하지만 눈앞에 펼쳐진 등교를 방해하는 수많은 장애물들! 
과연 눈송은 학교에 갈 수 있을까요 ?

-맵 곳곳을 돌아다니며 다양한 엔딩들을 수집하세요.
-인게임에서 귀여운 미니게임을 즐겨보세요.
-제작자가 손수 그린 엔딩그림을 앨범에서 확인하세요.
